Develop
-------
### Install Dependencies
```bash
pip install -r requirements.txt
```

### Run from sources
Run `server_run.py`

### Make
Run `make.bat`. ([tutorial](./docs/build.md))

Functions 
---------
* Account management.
* Server management.
* Travel diary.(travel map)
* Accounting function.
* System tray icon.
